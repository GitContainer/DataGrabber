# DataGrabber
This is a query tool that started years ago and is now being ported and extended to support Delphi10Seattle.

Connection topologies:
- UNIDAC (optional)
- ADO
- ZeosDBO (optional)
- FireDAC (todo)
- dbExpress

External dependencies:
- Spring4D
- DDuce
- Virtual Treeview
- ZeosDBO

Commercial components (optional):
- FastReport
- DevExpress
- UniDac
